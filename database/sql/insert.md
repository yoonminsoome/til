# insert

### 모든 컬럼에 값 삽입

```sql
insert into 테이블명 
values (값1, 값2, 값3, ...);
```
- 테이블의 컬럼 순서와 개수를 정확히 맞춰야 한다.

### 특정 컬럼만 지정해서 값 삽입

```sql
insert into 테이블명 (컬럼1, 컬럼2) 
values (값1, 값2);
```

- 나머지 컬럼은 DEFAULT 값을 적용하고 NULL 허용 시 NULL을 저장한다.

### 한 번에 여러 행에 값 삽입

```sql
insert into 테이블명 (컬럼1, 컬럼2) 
values 
    (값1, 값2), 
    (값3, 값4), 
    (값5, 값6);
```

- 한 번의 쿼리로 여러 행 삽입 가능
